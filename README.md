# KIRA ID - Universal Identity System

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000/connect

## Features

- Wallet connection with Phantom/Solflare/Backpack
- Telegram handle linking
- X/Twitter handle linking  
- Unified profile across all platforms
- Token balance reading
- Engagement tracking

## Environment Variables

```
NEXT_PUBLIC_SOLANA_RPC=https://api.mainnet-beta.solana.com
SUPABASE_URL=
SUPABASE_KEY=
TELEGRAM_BOT_TOKEN=
X_API_KEY=
```
